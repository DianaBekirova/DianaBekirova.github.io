Итого:

static

Иначе называется "баз позиционирования"

relative

Сдвигает элемент относительно текущего места

  1. Противоположные границы left/right(top/bottom)одновременно указать нельзя
  2. Окружающие элементы ведут себя так, будто элемент никуда не сдвигался.

absolute

Визуально переносит элемент на новое место.
новое место вычисляеться по координатам left/right/top/bottom относитильно ближайшего позиционированного родителя. Если такого родителя нет, то им считаеться окно.

  • Ширинав элемента по умолчанию устанавливается по содержимому.
  • Можно указать одноременно противоположные границы left/right(top/bottom). Элемент при этом растянется.
  • Окружающие элементы заполняют освободившиешься место.

fixed

Подвид абсолютного позиционирования, при котором элементпривязывается к кординатам окна, а не документа. При прокрутке остаётся на том же месте.
